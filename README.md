### Getting Started

Clone the repo -- install the dependencies and start the server.

```
npm install
npm start
open http://localhost:3000

```

### Dependencies

* React
* Webpack
* Websockets
* Express
* [babel-loader](https://github.com/babel/babel-loader)
* [webpack-dev-server](https://github.com/webpack/webpack-dev-server)
